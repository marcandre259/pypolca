# pypoLCA

Python bindings around a C++ reimplementation of the [poLCA](https://github.com/dlinzer/poLCA) R package for latent class analysis.

## Install

```bash
uv pip install -e ".[dev]"
```

## Credits

Drew A. Linzer and Jeffrey Lewis. 2011. "poLCA: an R Package for Polytomous Variable Latent Class Analysis." *Journal of Statistical Software* 42(10): 1-29. https://www.jstatsoft.org/v42/i10
